## Requirements

Before using this repository, you'll need:

* An Xbox 360 console
* A compatible **BadUpdate** setup
* A FAT32-formatted USB drive
* A supported entry point for the exploit

For official BadUpdate documentation, see the original project and community documentation.

## Tutorial

A full step-by-step tutorial is included in `TUTORIAL.txt`.

### Quick Overview

1. Download or clone this repository.
2. Open `TUTORIAL.txt` and read the instructions carefully.
3. Copy the required files to your USB drive or storage device.
4. Follow the steps in `TUTORIAL.txt` on your Xbox 360.
5. Make sure all files are placed in the correct folders before starting.

If you run into any issues, refer back to `TUTORIAL.txt` for the complete guide.
